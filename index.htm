Bem vindo ao sistema de cadastro para vacinação de animais

Este sistema foi desenvolvido para facilitar o cadastro dos animais que serão vacinados na cidade de Sertãozinho - PB. Através deste sistema, os tutores poderão cadastrar seus animais de forma rápida e prática, garantindo que todos os animais sejam vacinados e protegidos contra doenças.

Para cadastrar seu animal, basta clicar no link "Cadastre seu animal" no menu acima e preencher o formulário com as informações solicitadas. Após o cadastro, você receberá uma confirmação por e-mail com os detalhes da vacinação.

Agradecemos a sua colaboração em manter a saúde dos animais da nossa cidade!

Sobre o projeto

Este projeto foi desenvolvido por alunos do curso de Análise e Desenvolvimento de Sistemas da Universidade Cruzeiro do Sul como parte de um trabalho acadêmico. O objetivo do projeto é criar uma aplicação web simples e funcional para o cadastro de animais para vacinação, utilizando tecnologias como HTML, CSS e JavaScript.
